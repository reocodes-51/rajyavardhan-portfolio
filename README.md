##  About

A modern, responsive developer portfolio showcasing my projects, technical skills, experience, and achievements. Built with a premium UI, smooth animations, and optimized performance to reflect my journey as a software engineer.

---

## ✨ Features

- 🎨 Modern & Responsive UI
- ⚡ Built with Next.js & TypeScript
- 🎭 Framer Motion Animations
- 🌙 Premium Dark Theme
- 💼 Project Showcase
- 🛠️ Skills & Experience
- 📬 Contact Section
- 🚀 Deployed on Vercel

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, TypeScript, Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons, Lucide React
- **Deployment:** Vercel
- **Version Control:** Git & GitHub

## ⚙️ Getting Started

```bash
git clone https://github.com/reocodes-51/portfolio.git

cd portfolio

npm install

npm run dev
```

Open **http://localhost:3000** in your browser.
